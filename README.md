# Proyecto Teoría de Base de Datos 1

**Autor:** Fernando Hernández  
**Motor:** MariaDB  

---

## Descripción

Diseño del modelo de base de datos para un Sistema de Presupuesto Personal.

Incluye el modelo entidad–relación (ERD), el archivo DBML y el script SQL compatible con MariaDB.

---

## Entidades Principales

- Usuario  
- Presupuesto  
- Categoria  
- Subcategoria  
- PresupuestoDetalle  
- ObligacionFija  
- Transaccion  

---

## Archivos del Repositorio

- `.dbml` → Modelo diseñado en dbdiagram.io  
- `.sql` → Script de creación de tablas  
- ERD → Diagrama del modelo relacional
